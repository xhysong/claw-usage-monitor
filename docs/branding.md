# Branding

## Name
- Claw Monitor

## Icon direction
- Cartoon style: lobster claw (single pincer) + a small monitor/screen.
- Silhouette should be readable at 16–32px (menu bar + app icon small sizes).
- Prefer simple shapes + 2–3 main colors (e.g., orange/red claw + dark monitor + light screen).

## Deliverables
- App icon: 1024x1024 PNG (then Xcode will generate required sizes)
- Menu bar icon: monochrome template PDF/SVG or 18x18 / 20x20 / 22x22 PNG variants

## Notes
- Menu bar uses macOS template images (single color) for best appearance in light/dark mode.
